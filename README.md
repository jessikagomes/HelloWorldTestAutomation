## Selenium: Hello World Test-Automation:computer:

------

What you will find on this project: Java, Maven, Selenium, JUnit and Cucumber.

Our goal: looking for course on Iterasys page: https://iterasys.com.br/

#### :space_invader:Test Case

	Feature: Consulting course on the website
		Scenario: Happy path
	        Given I acces the Iterasys webpage
	        When I search for "teste de software"
	        And show a list of courses
	        And I click on Incio Rapido em Teste de software
	        Then I see the duration of the course is "60" hours
#### :space_invader:Prerequisite

Must have already installed on your computer:

- Java; - if you are not sure type `java -version` on your cmd
- Eclipse;

If you haven't, please check these links: [Java](https://www.java.com/pt-BR/download/ie_manual.jsp?locale=pt_BR) | [Eclipse](https://www.eclipse.org/downloads/)
